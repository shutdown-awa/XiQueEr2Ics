# XiQueEr2Ics
一个可以自动获取喜鹊儿课表并转换成.ics文件的工具

## 使用说明
1. 需要Node.js环境
2. 使用pip -r requirements.txt安装第三方Python库
3. xqe.py可以单独使用
4. api.py启动命令（请按照实际需求修改）：uvicorn main:app --host 0.0.0.0 --port 8000
5. 本代码仅针对本人所在学校特别优化，部分设置项请自行修改
